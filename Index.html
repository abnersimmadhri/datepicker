English

In this demo the default date picker is used.

  • The date picker default language is english.
  • The date picker has no public holidays configured.
  • Users are allowed to select weekends.
  • Default date format is : Month dd, yyyy



Français

In this demo the user is able to specify values for the date picker.

  • The id to bind the date picker to.
  • The id of the input field to bind to.
  • List of months in the users language.
  • Flag to disable weekends.
  • List of public holidays to disable.
  • Custom date format.
